# EfficientRBE-C-
基于论文《**Efficient registration-based encryption**》的python实现 ([https://github.com/ahmadrezarahimi/efficientRBE]) 翻译而来，在此感谢Gemini3 pro，太tm好用了，**不保证完全正确**
# 文件结构
- .vscode（vscode配置文件）
- BasicVersion（基础版本，不高效版本）
- EfficientVersion（高效版本）
- mcl-lib（mcl库，[https://github.com/herumi/mcl]）
# 运行环境
仅代表我自己的运行环境
- ubuntu 24.04 lts
- 下载好**SQLite3**库
- mcl库，就是mcl-lib
# 运行方式（vscode编辑器）
编译运行前，自己配置好需要用的插件，例如C/C++，以及自己搭建好C/C++环境。
1. 用vscode打开该文件夹
2. 点击左侧调试与运行
3. 在左侧选择要运行的任务即可
